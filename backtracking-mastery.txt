# Master Backtracking Template

```java
void backtrack(State state){

    if(base_condition){
        result.add(copy_of_state);
        return;
    }

    for(choice in choices){

        if(invalid_choice) continue;

        make_choice();

        backtrack(updated_state);

        undo_choice();   // backtrack
    }
}
```

# Core Thinking Model

Every backtracking problem must define:

| Element     | Meaning                         |
| ----------- | ------------------------------- |
| State       | current path / partial solution |
| Choices     | options available               |
| Constraints | rules to prune                  |
| Base case   | when solution is complete       |
